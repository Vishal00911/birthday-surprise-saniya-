# Birthday Surprise for Saniya

A minimal single-page birthday website with soft colors, floral accents, scroll animations, an interactive gift reveal, and a final surprise message.

## Edit the personal details

- In `index.html`, replace the paragraph in the message section with your own note.
- Replace `[YOUR NAME]` near the bottom with your name.
- Replace the photo placeholder in the surprise section with an image if you want to add a memory.

## Preview

Open `index.html` in a browser. The page is fully static and can be published with GitHub Pages.
